# L&C Mortgage Insights Chatbot

A Streamlit-based AI chatbot designed to answer questions about L&C mortgage business performance.

## Features
- Chat interface for business questions
- Focus on historic financial performance, trading conversions, and application numbers
- CSV upload for real data-backed answers
- Built with Python, Streamlit, and OpenAI

## Example Questions
- What were application volumes last month?
- How have conversions changed over time?
- Summarise the historic financial performance trend.
- Which month had the strongest application numbers?

## Tech Stack
- Python
- Streamlit
- OpenAI API
- python-dotenv
- pandas

## How to Run
1. Install dependencies:
   pip install -r requirements.txt
2. Add your API key to `.env`
3. Run:
   streamlit run app.py
